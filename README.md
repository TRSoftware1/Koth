---

| **Permission**      | **Description**                                      |
|----------------------|------------------------------------------------------|
| `koth.admin`         | Grants access to all admin commands for the plugin.  |
| `koth.admin.start`         | Permission to manually start a KOTH event.          |
| `koth.admin.stop`          | Permission to stop an ongoing KOTH event.           |
